- rm-0434
	- RMs:
		- RM 0434
	- LPTIM_ISR (also LPTIM_ICR and LPTIM_IER) don't alternate between input and output compare mode
	- LPTIM_ISR (also LPTIM_ICR and LPTIM_IER but always same names in fields)
		- has 7 fields
			- CMPM
			- ARRM
			- EXTTRIG
			- CMPOK
			- ARROK
			- UP
			- DOWN
	- LPTIM_CFGR without TRIGSEL extension
	- LPTIM_CR
		- has 5 fields
			- ENABLE
			- SNGSTRT
			- CNTSTRT
			- COUNTRST
			- RSTARE
	- LPTIM_OR
		- has 2 fields
			- OR_0
			- OR_1

	- no LPTIM_CCR1
	- no LPTIM_CFGR2
